# Changelog

All notable changes to this project are documented in this file.

## [0.1.1] - 2026-06-24

### Changed

- Prepared the repository and release metadata for submission to the default HACS store.

## [0.1.0] - 2026-06-24

### Added

- Initial standalone Home Connect oven card
- Automatic entity discovery by Home Assistant device ID
- Temperature, programme, progress, finish-time and device-status display
- Programme, temperature, timing and fast-preheat controls
- HACS packaging, Jenkins pipeline and GitHub Actions validation
